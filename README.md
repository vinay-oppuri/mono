# Mono

Mono is a Next.js application that lets users create custom AI agents and chat with them directly, similar to ChatGPT or Gemini. Each agent follows its own instructions, so users can create different assistants for different workflows.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 to view the app.

## Features

- Create and manage AI agents
- Start persistent chats with agents
- Save chat history
- Use custom instructions per agent
- Upgrade with Polar subscriptions

## Tech Stack

- Next.js
- TypeScript
- tRPC
- Drizzle ORM
- Better Auth
- Gemini
- Polar
- Tailwind CSS

## Deployment

Deploy easily with Vercel or another Next.js-compatible host.
